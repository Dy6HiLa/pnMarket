# pnMarket 1.0.0

## Возможности

- GUI аукциона со страницами, поиском, категориями и сортировкой.
- Продажа предмета из основной руки через `/ah sell <цена>`.
- Покупка выбранного количества предметов из лота.
- Мгновенное обновление всех открытых меню после продажи, покупки, возврата и забора товара.
- Просмотр лотов продавца и поиск по названию или Material ID.
- Возврат своего активного лота и забор истёкших товаров из раздела «Мои товары».
- Автоматическое истечение лотов через 24 часа.
- Настраиваемые категории по Material ID, части Material ID, названию предмета и съедобности.
- Ограничения минимальной и максимальной цены лота.
- Лимиты активных лотов для основной Vault-группы игрока.
- SQLite по умолчанию, MySQL и MongoDB.
- Русская локализация, редактируемые сообщения и проверка обновлений.

## Совместимость

- Paper 1.17–1.21.x, Java 17 или новее.
- Paper 1.16.5 работает в экспериментальном режиме.
- На 1.21 GUI использует современные свечи и мангровые вывески. На 1.16.5 они автоматически заменяются совместимыми предметами.
