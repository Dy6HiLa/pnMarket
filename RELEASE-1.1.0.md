# pnMarket 1.1.0

## Совместимость

- Один JAR для Paper 1.16.5–1.21.x.
- Требуется Java 17 или новее.
- Иконки GUI заменены на материалы, существующие в Minecraft 1.16.5.

## Без изменений

- Аукцион, поиск, сортировка, частичная покупка и мгновенное обновление открытых меню.
- SQLite, MySQL и MongoDB.
- Настраиваемые категории, лимиты лотов и ограничение цен.
- Vault-экономика, истечение лотов и раздел возврата товаров.
