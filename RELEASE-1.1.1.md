# pnMarket 1.1.1

## Paper 1.16.5

Поддержка Paper 1.16.5 добавлена в экспериментальном режиме. На новых версиях Minecraft GUI сохраняет современные иконки, а на 1.16.5 автоматически подбираются совместимые предметы.

Если на 1.16.5 появляется ошибка или некорректно выглядит GUI, отправьте в [Discord-поддержку](https://discord.gg/rRbzq6cnc6):

- версию Paper;
- версию Java;
- полный текст ошибки из консоли;
- действие, после которого возникла проблема.

## Изменения

- Добавлен runtime fallback иконок: на 1.21 используются свечи и мангровые вывески, на 1.16.5 - шерсть и дубовые вывески.
- GUI использует только ItemFlag, которые существуют в API 1.16.5.
- Аукцион, поиск, покупка, синхронизация GUI и базы данных работают без изменений.
