# pnMarket 1.0.1

Крупное обновление аукциона: отдельная торговая площадка за PlayerPoints, продажа наборов предметов и полностью обновлённый интерфейс.

## Донат-аукцион

- Добавлен отдельный аукцион за PlayerPoints с командой `/dah`.
- Донат-аукцион поддерживает продажу, поиск и просмотр товаров игроков:
  - `/dah sell <цена>` — выставить предмет из основной руки;
  - `/dah kit <цена>` — выставить набор предметов;
  - `/dah search <название>` — найти нужный лот;
  - `/dah show <игрок>` — посмотреть товары игрока.
- Между обычным и донат-аукционом можно переключаться прямо в главном меню.
- У каждого аукциона собственные лоты и отдельный раздел «Мои товары».
- PlayerPoints является необязательным дополнением: если плагин не установлен, обычный аукцион `/ah` продолжает работать.

## Наборы предметов

- Добавлены команды `/ah kit <цена>` и `/dah kit <цена>` для продажи нескольких предметов одним лотом.
- В набор попадают занятые слоты основного инвентаря. Максимальный размер настраивается параметром `kits.max-slots`.
- Перед покупкой игрок может открыть набор и посмотреть всё его содержимое в отдельном окне.
- Каждый набор получает собственный числовой идентификатор.
- В аукцион добавлена отдельная категория «Наборы».
- Внешний вид набора зависит от версии сервера: мешочек, шалкер или бочка выбираются автоматически из доступных материалов.

## Обновлённый GUI

- Добавлена анимация открытия аукциона.
- Заголовок меню теперь также обновляется с анимацией.
- Переработан дизайн категорий, сортировки, переключения аукциона, раздела «Мои товары» и окна просмотра набора.
- Добавлены новые кнопки навигации с пользовательскими текстурами голов.
- Категории, сортировка, страницы и элементы управления обновляются без повторного открытия всего меню.
- Расширена русская локализация названий предметов.
- Убрано принудительное курсивное начертание названий предметов.
- Обновлён внешний вид справки `/ah` и `/dah`, а сообщения об использовании команд приведены к единому оформлению.

## Новые настройки

- `donate-listing-price.minimum` и `donate-listing-price.maximum` — ограничения цены в донат-аукционе.
- `prefix-vault` — оформление стоимости в обычном аукционе.
- `prefix-playerpoints` — оформление стоимости в донат-аукционе.
- `{price}` — плейсхолдер суммы в обоих шаблонах цены.
- `kits.max-slots` — максимальное количество занятых слотов в одном наборе.
- `categories.bundles` — настройка категории наборов.

## Совместимость

- Paper 1.17–1.21.x и Java 17 или новее.
- Paper 1.16.5 остаётся экспериментальным режимом.
- Vault и совместимый плагин экономики необходимы для обычного аукциона.
- PlayerPoints необходим только для донат-аукциона `/dah`.
